1. Para este tópico escolhi uma visualização interativa sobre música: http://research.google.com/bigpicture/music/#
1.1 A mensagem que a visualização de propõe a passar é mostrar a popularidade dos gêneros musicais ao longo do tempo. Esta visualização nos possibilita fazer um hover para mostrar quais os principais artistas/bandas deste gênero e clicar em um determinado gênero musical para detalhar melhor melhor quais os sub-generos que são mais populares ao longo do tempo. Ao clicar novamente, possibilita ver as bandas deste sub-gênero, mas sempre como o foco na popularidade.
1.2 No eixo horizontal temos o ano, no eixo vertical tempos a popularidade. Inicialmente temos tons para representar os gêneros musicais, ao clicar temos saturação para representar os sub-gêneros e ao clicar novamente, temos a saturação para indicar as bandas/artistas.
1.3 Como a tarefa principal do gráfico é descobrir tendências ou padrões da popularidade dos gêneros + sub-gêneros + artistas ao longo dos anos, é utilizado o eixo horizontal para representar o ano, pois é o canal mais eficiente. Para identificar a popularidade de um gênero é utilizado o canal de área. Para identificar os diferentes gêneros, é utilizado o canal da cor.
